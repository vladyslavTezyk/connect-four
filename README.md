# Connect Four

A basic Connect Four game implementation for playing in the terminal.

## Installation

Clone the repository and install dependencies:

```bash
git clone <repository-url>
cd connect-four
pnpm install
```

## Running the Game

To start the game, run:

```bash
pnpm start
```

## How to Play

- Players take turns dropping pieces into columns
- The first player to connect four pieces horizontally, vertically, or diagonally wins
- Enter the column number (1-7) where you want to drop your piece

## Requirements

- Node
- pnpm

# Technology choices
- TypeScript for secure and maintainable code
- Node.js for terminal execution and future server-side execution
- A simple command-line interface for user interaction
- Monorepo structure for better organization and scalability
